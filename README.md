# CO2-Time-Series-Analysis-and-Forecast
Analysed atmospheric CO₂ concentrations in Hawaii by using time series analysis in R. Used trend analysis, seasonal decomposition, linear regression modelling and ACF analysis to make assumptions on the data. Created 120 and 240 month forecasts using Meta’s Prophet forecasting model to investigate future atmospheric CO₂ concentrations in Hawaii. 

All the files needed for this code is in the CO2_Folder. To open this project download the CO2_Folder and open the .Rproj file
